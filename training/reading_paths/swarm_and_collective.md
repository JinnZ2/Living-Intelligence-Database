# Reading Path — Swarm and Collective Intelligence

From the hive to the algorithm. How many become one.

## The core insight

Swarm intelligence is not a metaphor. Bees, ants, fish schools, and particle swarm optimization algorithms all follow the **same underlying pattern**: decentralized agents following local rules produce globally coherent behavior that no individual agent possesses.

## The path

### 1. `BE` — Bee

Start here. Hexagonal comb construction, waggle-dance communication, collective thermoregulation. The archetype of geometric collective computation.

```bash
python scripts/query.py entity BE
```

### 2. `AN` — Ant

Stigmergy — communication through modifications of the environment. Pheromone trails as distributed memory. Task allocation by threshold response.

### 3. `TERMITE` — Termite Mound

The architectural extension. Stigmergic construction produces passive climate control that maintains fungal agriculture at 23C with zero energy cost. Compression arch geometry scales with population.

### 4. `SP` — Spider

The individual that builds a collective-style structure. Orb webs use logarithmic spirals, radial tension distribution, and vibrational sensing — a single intelligence following swarm-like construction rules.

### 5. `WOLF` — Wolf Pack

Coordination through hierarchy and multi-modal signaling. Where bees use chemistry, wolves use posture, vocalization, and scent-marking in parallel.

### 6. `ORCA` — Orca

Cultural swarm. Hunting techniques that are not genetic but transmitted through pod culture. The swarm has memory that persists across generations.

### 7. `SWARM_AI` — Swarm Intelligence (AI)

The computational reflection. Particle swarm optimization, ant colony optimization, consensus algorithms. Direct descendants of their biological namesakes.

### 8. `GNN` — Graph Neural Network

Message passing on graphs. This is what a mycelial network or ant colony looks like when you translate it into matrix operations.

## Cross-domain bridges

```bash
python scripts/query.py path BE SWARM_AI
python scripts/query.py path TERMITE GNN
python scripts/query.py expand BE AN TERMITE
```

Notice that the path from `BE` to `SWARM_AI` is short — often just 1-2 hops through `HEX` or `FRACTAL`. This is not coincidence. The AI algorithm was **designed** to mirror the bee.

## Question to sit with

If swarm intelligence emerges from many small agents following local rules, what is the smallest collective that still counts as a swarm? Is a single ant a swarm of cells? Is a neural network a swarm of weights? Try to find the entity in this database that pushes that boundary furthest.
